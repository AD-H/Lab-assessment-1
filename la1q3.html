This is a heading under div tag

The HTML div tag (division) is a generic, block-level container used to group other HTML elements into sections or divisions on a webpage. It has no inherent visual effect or semantic meaning on its own until styled using CSS or manipulated with JavaScript.

HTML block-level elements structure the layout of a web page by starting on a new line and taking up the full available width. They create a "block" that pushes other elements below them.
